Development

Using Figma, Adobe, Python, and a Postgres database to develop the first version of Sodaro's real-time data analysis and buddy insight system, an AI-powered engine that pulls live API data from news sources and social media to decode market sentiment before the bell rings. By capturing early behavioral trends, Sodaro helps identify the domino effect behind major price movements by giving swing traders a sharper edge and everyday users a smarter view of what’s really moving the market.
